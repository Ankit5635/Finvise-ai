# 🏛 System Architecture

```text
                User
                  │
                  ▼
         React Frontend (Vercel)
                  │
         REST API Requests
                  │
                  ▼
        FastAPI Backend (Render)
                  │
   ┌──────────────┼───────────────┐
   │              │               │
   ▼              ▼               ▼
Stock Data     News APIs      AI Models
(yFinance)     RSS Feeds     Groq/Gemini
   │              │               │
   └──────────────┼───────────────┘
                  ▼
          AI Brief Generator
                  │
      ┌───────────┴───────────┐
      ▼                       ▼
   React UI             Video Generator
                       (gTTS + MoviePy)
                  │
                  ▼
           Download MP4
```
